# 英語ワークシート
## 元の英語ワークシート
https://github.com/manabuyoshikawa0109/english-worksheet/blob/next-js/app/page.tsx
## 大前提
直接変更したい旨の指示があった時を除き、元の英語ワークシートから以下を変更してはいけない
- HTML構造(クラス名は変わっても問題ない)
- CSS
- Reactの挙動(処理自体は変わっても問題ないが、最終的な挙動は全く同じになること)

## 修正したいこと
* 以下の処理を変更したい
* 【変更前】
  * サイドバーの「教材」アイコンをクリック
  * 「教科」ごとの「教材」を選択
  * キャンバス(ページ)がその教材作成用に切り替わる
* 【変更後】
  * 「Canva Clone」アイコン右側の「File」をクリック
  * 「英語」「国語」「算数」等、教科ごとに項目がある
  * 「英語」にカーソルを合わせると「ワークシート」「フラッシュカード」のように教材ごとに項目がある(イメージは「Canva Clone」アイコン右側の「File」をクリックした際に表示される「Create new design」)
  * 「ワークシート」をクリック
  * 画面リロード。URLは「http://localhost:5173/{教科名}/{教材名}」
  * キャンバス(ページ)がその教材作成用(ワークシート)に切り替わる。
  * File > 英語 > ワークシートが選択状態となっている
  * サイドバーの「Template」アイコンの上に、その教材作成用(ワークシート)のアイコンを表示。それをクリックすると教材作成用のツール(英語ワークシートの場合、文字のスタイル・罫線のスタイル・罫線の幅)が表示される
```
File
  ├─ 英語
  │  ├─ ワークシート
  │  └─ フラッシュカード
  ├─ 国語(準備中)
  ├─ 算数(準備中)
  └─ その他
    　└─ キャンバス(デフォルト。canva-cloneの元々の機能を表示。URLは「http://localhost」)
```
